# Как опубликовать статью на Habr

Черновик готов: `HABR_ARTICLE_DRAFT.md`. Ниже — пошаговая инструкция публикации.

---

## Шаг 1 — Войти на Habr

Перейти на **https://habr.com** и войти в аккаунт.

---

## Шаг 2 — Создать публикацию

Нажать **«Написать публикацию»** (кнопка в правом верхнем углу).

---

## Шаг 3 — Заполнить поля

**Заголовок:**
```
Claude Code Workflow: как заставить AI делать сложную работу без надзора
```

**Хабы (выбрать все три):**
- Искусственный интеллект
- Разработка под Linux/Unix (или «Open Source»)
- Инструменты разработчика

**Тэги:**
```
claude, claude-code, workflow, многоагентность, llm, anthropic, tutorial
```

---

## Шаг 4 — Вставить текст

Скопировать содержимое `HABR_ARTICLE_DRAFT.md` (без строки с пометкой «Черновик»).

Блоки кода Habr поддерживает в формате ` ```javascript ... ``` ` — они сохранятся как есть.

**Секцию «Комментарий к статье»** не вставлять — она обёрнута в HTML-комментарий `<!-- -->` и в финальный текст не войдёт.

---

## Шаг 5 — Обложка

Если система запросит обложку: загрузить любое изображение размером **1200×630 px**.
Простой вариант — скриншот веб-тренажёра (открыть https://nickscherbakov.github.io/ClaudeCodeManual/ и сделать скриншот).

---

## Шаг 6 — Превью и публикация

- Нажать **«Предпросмотр»** — убедиться, что блоки кода отображаются правильно
- Нажать **«Опубликовать»**

---

## Шаг 7 — Первый комментарий (важно!)

Сразу после публикации добавить первый комментарий к статье:

```
Курс полностью бесплатный и открытый (лицензия CC0).

• Интерактивный тренажёр: https://nickscherbakov.github.io/ClaudeCodeManual/
• GitHub (исходники + лабы + эталонные решения): https://github.com/NickScherbakov/ClaudeCodeManual

Если статья оказалась полезной — звезда на GitHub поможет другим её найти.
```

---

## После публикации

Поделиться ссылкой на статью в Telegram-каналах, используя шаблоны из `PROMO_TEMPLATES.md`.
